## 7. 상속 (피상속 설정)

### 7-1. 피상속인 지정 (`/mypage/inheritance`)

**위치**: `app/mypage/inheritance/page.tsx`

| 기능 | 내용 |
|------|------|
| 피상속인 현황 조회 | `GET /api/inheritance` — 이름/이메일/상태 표시 |
| 피상속인 지정 | `POST /api/inheritance` — 이메일 + 개인 메시지 입력 → 동의 요청 발송 |
| 피상속인 해제 | `DELETE /api/inheritance?id=` — 확인 모달 포함 |
| 페르소나별 피상속 허용 토글 | `PATCH /api/inheritance` — 피상속 시 이전될 페르소나 선택 |
| 상태값 | `none` / `pending` (초대 중) / `accepted` (동의 완료) / `rejected` (거부됨) |

### 7-2. 피상속 수락 (`/mypage/inheritance-accept`)

**위치**: `app/mypage/inheritance-accept/page.tsx`

| 기능 | 내용 |
|------|------|
| 요청 목록 조회 | `GET /api/inheritance/consent` |
| 수락 | `POST /api/inheritance/consent` (action: accept) — 확인 모달 |
| 거절 | `POST /api/inheritance/consent` (action: reject) — 확인 모달 |
| 처리된 요청 내역 | 상태별 표시 (수락됨/거절됨/취소됨) |
| 수락 시 권한 | 챗봇 운영/대화 내역 열람(읽기전용)/설정 수정/배포 URL 관리 |

### 7-3. React page-client 내 피상속 요약 표시

프로필 탭 하단에 피상속인 이름/이메일/상태/초대일/허용 챗봇 수 요약 표시 + `/mypage/inheritance` 관리 링크.
