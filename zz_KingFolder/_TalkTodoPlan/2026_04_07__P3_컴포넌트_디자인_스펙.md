## 1. Button

### 1-1. 구조 (ASCII)

```
Primary:
┌─────────────────────────────┐
│  [아이콘?]  레이블 텍스트   │
└─────────────────────────────┘

Ghost:
┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐
│  [아이콘?]  레이블 텍스트   │
└ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘

Icon-only:
┌──────┐
│  ✕  │
└──────┘
```

### 1-2. 크기 규격

| 크기 | height | padding-x | font-size | icon-size | border-radius |
|------|--------|-----------|-----------|-----------|---------------|
| sm   | 32px   | 12px      | 13px      | 14px      | var(--radius-md) = 8px |
| md   | 40px   | 16px      | 14px      | 16px      | var(--radius-md) = 8px |
| lg   | 48px   | 24px      | 16px      | 18px      | var(--radius-lg) = 12px |

### 1-3. 4종 변형

#### Primary
```css
.btn-primary {
  background: var(--color-primary);
  color: var(--color-text-inverse);
  border: none;
  font-weight: 600;
  letter-spacing: -0.01em;
}
.btn-primary:hover  { background: var(--color-primary-hover); box-shadow: var(--shadow-md); transform: translateY(-1px); }
.btn-primary:active { background: #3A2BC7; transform: translateY(0); box-shadow: none; }
```
Tailwind: `bg-[#5E4BFF] text-white font-semibold px-4 py-2.5 rounded-lg hover:bg-[#4A39E0] hover:shadow-md hover:-translate-y-px active:translate-y-0 transition-all duration-200`

#### Secondary
```css
.btn-secondary {
  background: var(--color-primary-light);  /* #EEF0FF */
  color: var(--color-primary);
  border: 1.5px solid var(--color-border);
  font-weight: 600;
}
.btn-secondary:hover  { background: #E0DCFF; border-color: var(--color-primary); }
.btn-secondary:active { background: #D5D0FF; }
```
Tailwind: `bg-[#EEF0FF] text-[#5E4BFF] border border-[#E5E4F0] font-semibold px-4 py-2.5 rounded-lg hover:bg-[#E0DCFF] hover:border-[#5E4BFF] transition-all duration-200`

#### Ghost
```css
.btn-ghost {
  background: transparent;
  color: var(--color-text-secondary);
  border: 1.5px solid var(--color-border);
  font-weight: 500;
}

```
Tailwind: `bg-transparent text-[#6B6884] border border-[#E5E4F0] font-medium px-4 py-2.5 rounded-lg hover:bg-[#F1F0F9] hover:text-[#1A1830] hover:border-[#C9C7E8] transition-all duration-200`

#### Danger
```css

  background: var(--color-danger);
  color: white;
  border: none;
  font-weight: 600;
}
.btn-danger:hover  { background: var(--color-danger-hover); box-shadow: 0 4px 16px rgba(239,68,68,0.3); }
```

### 1-4. 공통 상태

| 상태 | 처리 |
|------|------|
| **disabled** | `opacity: 0.4; cursor: not-allowed; pointer-events: none;` |
| **loading** | 텍스트 숨기고 spinner SVG 표시 (16px, white). `cursor: wait;` width 고정으로 레이아웃 shift 방지 |
| **icon-left** | gap: 6px; icon + label 수평 배치 |
| **icon-only** | padding-x = padding-y (정사각형), aria-label 필수 |

### 1-5. 아이콘 버튼 변형
```
sm icon-only: 32×32px, radius 8px
md icon-only: 40×40px, radius 8px
lg icon-only: 48×48px, radius 12px
```
Tailwind (md): `w-10 h-10 flex items-center justify-center rounded-lg`
