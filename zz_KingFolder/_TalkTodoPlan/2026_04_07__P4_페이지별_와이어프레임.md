## 공통 설계 원칙

| 원칙 | 내용 |
|------|------|
| **다크/라이트 동시 지원** | 모든 와이어프레임은 CSS 토큰 기반으로 다크/라이트 모드 모두 지원. 좌측 색상값은 다크 모드 기준 참고값. |
| **컬러 표기법** | [P] = Primary 퍼플 / [A] = Accent 앰버 |
| **컴포넌트 표기법** | BtnPrimary, BtnSecondary, SkillCard, BotCard, JobCard, PostCard, Input, Badge |
| **레이아웃 표기** | 박스 문자로 경계 표시, 반응형 지원 |
| **반응형 기준** | 데스크탑: 1024px+ / 태블릿: 768px+ / 모바일: ~767px |
