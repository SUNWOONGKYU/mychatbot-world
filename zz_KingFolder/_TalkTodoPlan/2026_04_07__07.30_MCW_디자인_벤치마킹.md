## 2. 핵심 디자인 패턴 분석

### 네비게이션 패턴

```
TOP BAR 독점 구조:
- 모든 서비스가 상단 수평 내비게이션 바 사용 (예외 없음)
- 랜딩 페이지: 상단바 / 앱 내부: 좌측 사이드바 (이중 구조)
- 평균 메뉴 항목: 4~7개 (초과 시 메가메뉴/드롭다운)
- 우측 고정 CTA 버튼이 공통 패턴 (Get Started / Free Trial / Book Demo)
```

### 빌더 UI 패턴 분포

```
캔버스/노드 기반: Botpress, Voiceflow, Dify, Jasper, Copy.ai (5개) — 복잡한 플로우
비주얼 플로우 에디터: Typebot (45+ 블록, 간소화 캔버스)
단계별 위저드 (3-Step): Dante AI, CustomGPT — 빠른 진입 강조
폼+모듈 복합: Channel.io, 카카오 i 빌더
```

### 컬러 트렌드

```
다크 모드 퍼스트: Botpress (퍼플-마젠타), Copy.ai (퍼플-틸)
라이트 모드 + 그라데이션 포인트: Chatbase, Chatfuel, Channel.io
단색 포인트 미니멀: Typebot, CustomGPT, Gumroad
퍼플 계열 지배: Botpress #E2AEEF / Channel.io #5E56F0 / Copy.ai #693EE0 — "AI=퍼플" 트렌드
```

### 마켓플레이스 카드 설계 패턴

수집된 주요 패턴 (Botpress Hub, Chatfuel Template, Dify Marketplace 기준):

```
필수 요소: 섬네일/아이콘 + 제목 + 카테고리 배지
권장 요소: 설명 1~2줄 + 사용 횟수/다운로드 수 + 작성자
차별화 요소: 가격 배지 (무료/유료), 인기 라벨, 호환 채널 표시
그리드: 3열 (데스크탑) → 2열 (태블릿) → 1열 (모바일)
```

### 랜딩 페이지 Hero 공식

```
[강렬한 1줄 헤드라인]
```

## 3. MCW 적용 디자인 방향 제안 (7가지)

### 제안 1: "3-Step 위저드 + 캔버스 이중 진입점" 빌더 구조

**벤치마크 근거**: Dante AI/CustomGPT의 3단계 위저드(빠른 진입)와 Botpress/Voiceflow의 캔버스(고급 제어)가 각기 다른 사용자 층을 공략한다.

**MCW 적용안**:
- **초보자 진입**: "5분 만에 챗봇 만들기" — 3단계 위저드 (데이터 연결 → 스킬 장착 → 배포)
- **고급 사용자**: "캔버스 에디터" 전환 버튼으로 플로우 노드 편집기 제공
- 두 모드를 동일한 프로젝트에서 전환 가능하게 설계

### 제안 2: "퍼플 + 앰버(골드)" 브랜드 컬러 확립

**벤치마크 근거**: 퍼플 계열이 AI SaaS 시장의 지배적 색상 (Botpress #E2AEEF, Channel.io #5E56F0, Copy.ai #693EE0). 그러나 MCW는 "수익 창출" 가치가 핵심 — 골드/앰버 포인트가 차별화.

**MCW 적용안**:
- Primary: 딥 퍼플 (#5E4BFF 계열) — AI/기술 신뢰감
- Accent: 앰버/골드 (#F59E0B 계열) — 수익/크리에이터 에너지
- Background: 다크 모드 지원 (경쟁사 절반이 다크 모드 퍼스트)
- 그라데이션: 퍼플→블루 (AI 공간감), 앰버→오렌지 (수익 강조 섹션)

### 제안 3: "상단바 + 앱 내 좌측 사이드바" 이중 네비게이션 구조

**벤치마크 근거**: 모든 서비스가 랜딩은 상단바, 앱 내부는 좌측 사이드바. 이 패턴이 업계 표준.

**MCW 적용안**:
- 랜딩/마케팅 페이지: 상단바 (Logo + 스킬스토어 + 가격 + 로그인 + "무료 시작" CTA)
- 앱 대시보드: 좌측 사이드바 (챗봇 목록 / 스킬스토어 / 분석 / 수익 / 설정)
- 사이드바 항목: 5~7개로 제한, 아이콘+텍스트 조합

### 제안 4: 스킬스토어 카드 = "앱스토어 문법" 적용

**벤치마크 근거**: Botpress Hub 3열 그리드, Chatfuel 템플릿 라이브러리, Dify Marketplace.

**MCW 스킬스토어 카드 스펙 제안**:
```
┌─────────────────────────┐
│  [카테고리 컬러 아이콘]  │  ← 16x16 or SVG
│  스킬 이름              │  ← Bold, 1줄
│  설명 2줄 텍스트        │  ← 그레이, 2줄 클램프
│  ─────────────────────  │
│  [작성자] ★4.8 (234)   │  ← 소형 텍스트
│  [무료] or [₩9,900]   │  ← 우측 정렬 배지
└─────────────────────────┘
```
- 그리드: 데스크탑 4열 / 태블릿 3열 / 모바일 2열 (앱스토어 밀도)
- 필터: 카테고리 탭 (고객응대 / 마케팅 / 커머스 / 리서치 / 한국어 특화) + 가격 (무료/유료/모두) + 정렬 (인기/최신/평점)

### 제안 5: 랜딩 Hero = "수익 숫자" 전면 배치

**벤치마크 근거**: Gumroad "$2,109,451/주 크리에이터 수익" 배치가 가장 강력한 크리에이터 마켓플레이스 신뢰 신호. Chatbase "10,000+ 기업", Copy.ai "17,000,000 사용자" — 모두 숫자 기반 사회적 증거.

**MCW 적용안**:
```
Hero 구성 (상단→하단):
① "누구나 5분 만에 AI 챗봇으로 수익을" (1줄 헤드라인)
② "스킬 장착 → 챗봇 배포 → 수익 창출" (3단계 서브카피)
③ [무료로 시작하기] [데모 보기] (이중 CTA)
④ ★실시간 카운터: "지금까지 ₩xxx,xxx 수익 창출됨" (신뢰 지표)
⑤ 챗봇 에디터 스크린샷 or 인터랙티브 데모
```

### 제안 6: 가격 티어 = "크리에이터 성장 단계" 내러티브 매핑

**벤치마크 근거**: Chatbase 5티어 + "Popular" 강조 + 연간 20% 할인 + 애드온 카드 패턴이 전환율 최적화 표준.

**MCW 적용안**:
```
Free    → "탐험가" (챗봇 1개, 스킬 3개 무료)
Starter → "창작자" (챗봇 5개, 수익 창출 개방) — ₩9,900/월
Pro     → "전문가" (무제한 챗봇, 스킬 전체) — "Most Popular"
Business → "기업/팀" (멀티시트, 화이트라벨) — ₩99,000/월
```
- 각 티어명에 한국 크리에이터 친화 용어 사용
- 연간 결제 배너 (20% 할인 강조)

### 제안 7: 모바일 = "하단 탭바" 우선 전략

**벤치마크 근거**: 모든 서비스가 모바일에서 햄버거 메뉴 사용 — 그러나 이는 챗봇 "관리" 중심 UX. MCW는 한국 모바일 유저 비중이 높아 하단 탭바가 더 적합.

**MCW 모바일 전략**:
- 웹앱 (PWA 대응): 하단 탭바 5개 (홈 / 챗봇 / 스토어 / 분석 / 내 수익)
- 데스크탑: 좌측 사이드바로 자동 전환
- 터치 타겟: 최소 48px (iOS/Android 권장)
- 채널톡처럼 padding을 모바일에서 극적으로 줄임 (140px → 16px)

## 5. MCW 디자인 총평 및 우선순위 실행 계획

### 핵심 차별화 포인트

경쟁사 분석 결과, MCW만의 차별점은:
1. **스킬 마켓플레이스** — 챗봇 빌더에서 스킬을 사고파는 생태계 (경쟁사 대부분 없음)
2. **수익 창출 전면 강조** — 크리에이터 경제 관점 (Gumroad 문법 + AI 챗봇 빌더)
3. **한국어 최적화** — 채널톡처럼 한국 비즈니스 맥락에 맞는 UX

### 실행 우선순위

```
P0 (즉시):
  - 브랜드 컬러 확정 (퍼플 + 앰버 골드 팔레트)
  - 랜딩 Hero 카피 + 수익 카운터 배치
  - 스킬스토어 카드 컴포넌트 설계

P1 (1~2주):
  - 3단계 위저드 빌더 플로우 완성
  - 가격 티어 4단계 + 애드온 카드 페이지
  - 모바일 하단 탭바 구현

P2 (2~4주):
  - 다크 모드 완전 지원
  - 인터랙티브 랜딩 배경 애니메이션
  - 역할/업종별 진입 경로 분기
```
