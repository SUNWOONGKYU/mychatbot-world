## 0. 설계 원칙

1. **4대 메뉴 확정** — Birth / Skills / Jobs / Community (Learning은 마이페이지 "챗봇 학습" 탭으로 통합, PO 승인 2026-04-07)
2. **업계 표준 이중 구조** — 랜딩/마케팅 페이지는 상단바, 앱 내부는 좌측 사이드바
3. **사용자 여정 우선** — 비로그인 방문 → 가입 → 챗봇 생성 → 스킬 장착 → 수익 창출
4. **모바일 하단 탭바** — 한국 모바일 유저 비중 고려, 햄버거 메뉴 대신 하단 탭
5. **메뉴 수 제한** — 최대 7개 이내 (인지 부하 최소화)

## 2. 데스크탑 네비게이션 구조

### 2-1. 마케팅 상단 GNB (비로그인 상태)

```
┌─────────────────────────────────────────────────────────────────────┐
│  [MCW 로고]   스킬스토어   가격   커뮤니티   블로그        [로그인] [무료로 시작하기 ▶] │
└─────────────────────────────────────────────────────────────────────┘
```

| 위치 | 항목 | 연결 | 비고 |
|------|------|------|------|
| 좌측 | MCW 로고 | `/` | 클릭 시 랜딩 이동 |
| 중앙 좌 | 스킬스토어 | `/store` | 비로그인도 탐색 가능 |
| 중앙 | 가격 | `/pricing` | 플랜 비교 페이지 |
| 중앙 | 커뮤니티 | `/community` | 공개 게시판 일부 노출 |
| 중앙 | 블로그 | `/blog` | 컨텐츠 마케팅 |
| 우측 | 로그인 | `/login` | 텍스트 링크 |
| 우측 | 무료로 시작하기 | `/signup` | 강조 CTA 버튼 (Primary 컬러) |

**드롭다운 — 스킬스토어**
```
스킬스토어 ▾
├── 전체 스킬 목록
├── 인기 스킬 Top 20
├── 카테고리별 탐색
│   ├── 고객응대
│   ├── 마케팅·판매
│   ├── 커머스
│   ├── 리서치·분석
│   └── 한국어 특화
└── 스킬 판매하기 (크리에이터 CTA)
```

- 너비: 240px (기본) / 접기 버튼 클릭 시 64px 아이콘 전용 모드
- 고정(sticky): 스크롤 시 사이드바 고정, 컨텐츠 영역만 스크롤

## 3. 태블릿 네비게이션 구조 (768px ~ 1023px)

### 3-1. 마케팅 페이지 (태블릿)

```
┌──────────────────────────────────────────┐
│ [MCW 로고]    스킬스토어  가격  [시작하기] [≡] │
└──────────────────────────────────────────┘
```
- "커뮤니티", "블로그" → 햄버거 드로어(≡)로 이동
- 주요 CTA와 핵심 2개 메뉴만 상단 노출
- 햄버거 드로어: 오버레이 슬라이드 (우측에서 진입)

### 3-2. 앱 내부 (태블릿)

```
[64px 아이콘 전용 사이드바 + 호버 시 툴팁 표시]
```
- 사이드바 기본 상태: 64px (아이콘만)
- 아이콘 호버 시: 툴팁으로 메뉴명 표시
- 아이콘 클릭 시: 해당 페이지 이동 (서브메뉴 없음, 상단 브레드크럼으로 대체)
- 또는: 사이드바 완전 숨김 + 상단바에 5대 메뉴 탭으로 표시

**태블릿 권장 방식 — 상단바 탭 전환**
```
┌──────────────────────────────────────────────────┐
│ [≡]  Home  Birth  Skills  Jobs  Community  [👤]  │
└──────────────────────────────────────────────────┘
```
- Learning 제거 (마이페이지 "챗봇 학습" 탭으로 통합 — PO 2026-04-07)
- 좌측 햄버거: 기타 페이지
- 우측 아바타: MyPage(8탭) + 관리자(권한자) + 설정

## 6. 4대 메뉴 밖 페이지 — 접근 경로

- **접근**: 사이드바 최상단 / 모바일 탭바 1번 / 로그인 후 자동 리다이렉트

### 6-2. MyPage (내 페이지) — 8탭 구조 확정
- **접근**: 사이드바 하단 "👤 마이페이지" / 상단바 아바타 드롭다운 / 모바일 상단 헤더 아바타
- **위치**: `/mypage`
- **8개 탭** (PO 확정 2026-04-07):
  - `/mypage/profile` — 탭1. 프로필 관리 (닉네임/이메일/아바타/알림)
  - `/mypage/chatbot` — 탭2. 챗봇 관리 (목록·페르소나·배포·복제·내보내기)

  - `/mypage/skills` — 탭4. 스킬 관리 (다운로드 스킬 목록·마켓 등록)
  - `/mypage/operations` — 탭5. 챗봇 운영 관리 (구직·구봇·수익·정산)
  - `/mypage/inheritance` — 탭6. 상속 (페르소나 단위 상속 설정)
  - `/mypage/credits` — 탭7. 크레딧/결제 (잔액·충전·무통장입금·내역)
  - `/mypage/security` — 탭8. 계정 보안 (비밀번호·로그아웃·계정삭제)

### 6-3. Admin (관리자 대시보드) — 8섹션 구조 확정
- **접근**: 사이드바 하단 "🔐 관리자" (관리자 권한 사용자에게만 노출)
- **위치**: `/admin`
- **8개 섹션** (PO 확정 2026-04-07):

  - `/admin/notice` — 섹션2. 공지사항 관리
  - `/admin/members` — 섹션3. 회원 관리
  - `/admin/payments` — 섹션4. 입금/결제 관리 (무통장 승인·크레딧 충전)
  - `/admin/bots` — 섹션5. 챗봇 관리
  - `/admin/skills` — 섹션6. 스킬 관리 (공식 스킬 등록·회원 스킬 검수)

### 6-4. 수익 현황 (Earnings)
- **접근**: 마이페이지 탭5 "챗봇 운영 관리" 내 수익 섹션 또는 Home 대시보드 수익 위젯

- **서브 페이지**: 계정 / 알림 / 보안 / 연동 서비스 / 플랜

- **로그인 후**: 현재 구독 티어 강조 표시

### 6-7. Business (사업자 전용) — 추후 추가

## 7. 사용자 여정별 네비게이션 동선

### 여정 1: 첫 방문자 → 가입
```
랜딩(/) → [무료로 시작하기] → 회원가입(/signup) → 대시보드(/dashboard)
         → 가격 페이지(/pricing) 탐색 → 플랜 선택 → 가입
```

### 여정 2: 가입 → 챗봇 생성
```
대시보드 → 사이드바 "Birth" → 챗봇 생성 위저드 (3단계)
Step 1: 목적 선택 (고객응대 / 판매 / 상담)
Step 2: 스킬 장착 → 스킬스토어 연결
Step 3: 배포 + 공유 링크 생성
```
