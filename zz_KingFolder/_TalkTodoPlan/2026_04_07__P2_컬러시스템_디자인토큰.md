## 0. 설계 원칙

| 원칙 | 내용 |
|------|------|
| **다크/라이트 동시 지원** | 다크와 라이트 모두 완벽 지원. 기본값은 시스템 설정 또는 사용자 선택. (next-themes `defaultTheme="system"`) |
| **CSS 변수 → Tailwind 매핑** | 모든 색상은 CSS 변수로 정의 후 tailwind.config.ts에서 참조 |
| **RGB 포맷** | `rgb(var(--token) / <alpha>)` 패턴으로 Tailwind alpha 채널 연동 |
| **용도 주석 필수** | 변수마다 어디에 쓰는지 명시 |
| **퍼플 + 앰버 이중 정체성** | Primary(퍼플) = AI/기술, Accent(앰버) = 수익/크리에이터 |

## 3. 타이포그래피 스케일

### 3-1. 폰트 선택

**결정: Pretendard (한글/영문 공용) + 폴백**

Pretendard를 선택한 이유:
- 한글/영문 모두 커버. Inter를 별도 로드할 필요 없음.
- Apple SD Gothic Neo, Noto Sans KR 대비 획 굵기 제어가 정밀함.
- 라이선스: SIL OFL 1.1 (무료, 상업 가능).
- Channel.io가 Pretendard 계열 사용.

```css
/* globals.css에 추가 — Google Fonts CDN 또는 로컬 */
@import url('https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/variable/pretendardvariable.css');

:root {
  --font-sans: 'PretendardVariable', 'Pretendard', -apple-system,
               BlinkMacSystemFont, 'Apple SD Gothic Neo',
               'Malgun Gothic', 'Noto Sans KR', sans-serif;
  --font-mono: 'JetBrains Mono', 'Fira Code', 'Consolas',
               'Courier New', monospace;
}
```

| `--text-sm`   | `0.875rem` | `14px` | `1.5`   | `0.005em` | 보조 텍스트, 라벨 |
| `--text-base` | `1rem`     | `16px` | `1.625` | `0`       | 본문 기본 |
| `--text-lg`   | `1.125rem` | `18px` | `1.625` | `-0.005em`| 카드 제목, 리드 텍스트 |
| `--text-xl`   | `1.25rem`  | `20px` | `1.5`   | `-0.01em` | 소제목 |
| `--text-2xl`  | `1.5rem`   | `24px` | `1.375` | `-0.015em`| 섹션 제목 |

## 4. 간격/라운딩/그림자 토큰

### 4-1. Spacing (4px 기반)

```css
:root {
  --space-0:   0;

  --space-0_5: 0.125rem;  /* 2px */
  --space-1:   0.25rem;   /* 4px */
  --space-1_5: 0.375rem;  /* 6px */
  --space-2:   0.5rem;    /* 8px */
  --space-2_5: 0.625rem;  /* 10px */
  --space-3:   0.75rem;   /* 12px */
  --space-4:   1rem;      /* 16px */
  --space-5:   1.25rem;   /* 20px */
  --space-6:   1.5rem;    /* 24px */
  --space-8:   2rem;      /* 32px */
  --space-10:  2.5rem;    /* 40px */
  --space-12:  3rem;      /* 48px */
  --space-16:  4rem;      /* 64px */
```

**모바일 섹션 패딩 전략** (Channel.io 참고):
- 데스크탑: `py-24` (96px)
- 태블릿: `py-16` (64px)
- 모바일: `py-8` (32px) — 극적 축소

### 4-2. Border Radius

Chatfuel 스타일의 넉넉한 라운딩 (32~40px) 참고. MCW는 카드는 넉넉하게, 버튼은 중간.

```css
:root {
  --radius-none: 0;
  --radius-sm:   0.25rem;   /* 4px — 인풋, 작은 배지 */
  --radius-md:   0.5rem;    /* 8px — 보조 버튼 */
  --radius-lg:   0.75rem;   /* 12px — Primary 버튼, 작은 카드 */
  --radius-xl:   1rem;      /* 16px — 일반 카드 */
  --radius-2xl:  1.25rem;   /* 20px — 큰 카드, 모달 */
  --radius-3xl:  1.5rem;    /* 24px — Hero 데모 패널 */
  --radius-4xl:  2rem;      /* 32px — 랜딩 섹션 카드 (Chatfuel 스타일) */
  --radius-full: 9999px;    /* 완전 원형 — 아바타, pill 배지, FAB */
}
```
