|---|------|------|

## 마이페이지 8개 탭

### 탭1. 프로필 관리
- 닉네임, 이메일, 가입일 표시
- 프로필 아바타 업로드
- 알림 설정

### 탭2. 챗봇 관리
- 내 챗봇 목록 (카드 뷰) + 작성 중 초안 카드
- 챗봇별:
  - URL 패널 (배포 URL + QR 코드 생성 + URL 복사)
  - 페르소나 관리 (최대 10개) — 추가/삭제/편집
  - AI 인사말 자동 생성, FAQ 자동 생성
  - per-persona 툴 6종 패널:
    1. 대화 로그 뷰어
    2. KB 지식베이스 (업로드/FAQ/Obsidian)
    3. 스킬 장착/해제 (마켓에서 받은 스킬을 이 봇에 장착)
    4. 챗봇스쿨 (학습 현황)
    5. 커뮤니티 활동 내역
    6. 챗봇 설정 (AI 모델/온도/DM 보안정책 3단계)
  - 챗봇 복제 ("80% 같고 20% 다른 챗봇" 만들기)
  - 챗봇 내보내기/가져오기 (JSON 백업·복원)
  - 봇 삭제

### 탭3. 챗봇 학습 (구 Learning 통합)
- KB 주입 (파일 업로드 / 텍스트 입력 / Obsidian .md)
- Wiki-e-RAG 연동:
  - Wiki 자동 생성 (Ingest)
  - Wiki 페이지 목록/편집
  - Wiki 그래프 뷰 (연결 시각화)
  - Wiki Lint (품질 점검)
  - 축적 현황 (복리 성장 지표)
- FAQ 추가/편집
- 전체 챗봇의 학습 현황 한눈에 보기

### 탭4. 스킬 관리
- 스킬 마켓에서 다운로드 받은 내 스킬 목록
- 스킬 활성/비활성 토글
- 어떤 챗봇에 장착했는지 표시
- 스킬 제거
- **스킬 마켓에 내 스킬 등록하기** (스킬 제작 → 마켓 등록)

### 탭5. 챗봇 운영 관리
- **구직 (내 챗봇 올리기)**:
  - 내 챗봇을 구봇구직에 등록
  - 등록 공고 관리 (모집중/마감)
  - 지원자(고용자) 관리
  - 수익 유형 설정 (사용자 정의 — 유형명 직접 입력)
  - 유형별 단가
  - 수익 현황 (일/주/월)
  - 정산 요청
- **구봇 (다른 챗봇 고용하기)**:
  - 고용한 챗봇 목록
  - 고용 계약 관리 (기간/단가/조건)
  - 고용 챗봇 성과 확인
  - 결제/비용 내역
- **운영 통계**:
  - 대화 건수 / 만족도
  - 구직 수익 vs 구봇 비용

### 탭6. 상속
- **페르소나 단위** 상속 설정
- 페르소나별 피상속인 지정 (다른 사람도 가능)
- 페르소나별 **제외 항목 선택** (대화로그/KB/스킬/크레딧/수익설정/특정 페르소나)
- 상속 동의 요청 발송
- 상속 수락 페이지 (/mypage/inheritance-accept)
- 상속 상태 확인

### 탭7. 크레딧/결제
- 크레딧 잔액 표시
- 4개 패키지: **3만 / 5만 / 10만 / 기타(직접 입력)**
- **할인 없음** (표시 금액 = 결제 금액)
- 무통장 입금 신청 (Anthropic UI 참고)
- 사용 내역 / 충전 내역

### 탭8. 계정 보안
- 비밀번호 변경
- 로그아웃
- 계정 삭제

### + 알림센터 (탭 아님, 상단 헤더 벨 아이콘)
- 상속 동의 요청
- 입금 완료
- 스킬 업데이트
- 공지사항
- 구봇구직 매칭 알림

## 변경 이력

| 항목 | 변경 전 | 변경 후 | 사유 |
|------|--------|--------|------|
| 5대 메뉴 | Birth/Learning/Skills/Jobs/Community | Birth/Skills/Jobs/Community (4대) | Learning은 마이페이지 통합 |
| 크레딧 패키지 | 5천/1만/3만/5만 | 3만/5만/10만/기타 | 최소 3만원 이상 |
| 할인 | 10%/20%/30% | 없음 | PO 지시 |
| 수익 유형 | 토글 4개 고정 | 사용자 정의 (직접 입력) | 자기가 분석하고 싶은 기준으로 |
| 상속 단위 | 챗봇 단위 | 페르소나 단위 | PO 지시 |
| 상속 제외 | 없음 | 제외 항목 선택 가능 | PO 지시 |
| 구봇 기능 | 없음 | 다른 챗봇 고용 관리 | 구봇구직의 "구봇" 부분 누락 보완 |

| 챗봇 복제 | 없음 | 추가 | 80% 같고 20% 다른 챗봇 만들기 |
