# User Testing Plan

베타 테스트 계획 문서입니다.

---

## 🎯 목표

10명의 베타 테스터를 모집하여 실제 사용 피드백을 수집합니다.

---

## 👥 참가자 선정

**모집 인원**: 10명

**기준**:
- 다양한 사용 사례 (비즈니스, 학습, 건강 등)
- 기술 수준 다양성 (초급 ~ 고급)
- 연령대 다양성

---

## 📋 테스트 시나리오

### 1. 온보딩 (5분)
- 회원가입
- 페르소나 선택
- 첫 대화 시작

### 2. 채팅 기능 (15분)
- 여러 페르소나와 대화
- 감정 표현 확인
- 3D 아바타 반응 확인

### 3. 고급 기능 (10분)
- 음성 입력 (선택)
- 대화 검색
- 페르소나 전환

---

## 📊 수집 데이터

### 정량적 데이터
- 작업 완료 시간
- 클릭 수
- 오류 발생 횟수
- 페이지 로딩 시간

### 정성적 데이터
- 사용 편의성 (1-5)
- 만족도 (1-5)
- 개선 의견
- 버그 리포트

---

## 📝 설문 문항

1. 전반적인 만족도는? (1-5)
2. 가장 유용했던 기능은?
3. 가장 불편했던 점은?
4. 3D 아바타가 도움이 되었나요?
5. 개선이 필요한 부분은?
6. 추가로 원하는 기능은?

---

## 🔄 프로세스

1. **모집** (1주): 베타 테스터 모집
2. **온보딩** (3일): 테스트 가이드 제공
3. **테스트** (1주): 실제 사용
4. **피드백** (3일): 설문 및 인터뷰
5. **분석** (3일): 결과 분석 및 우선순위

---

**목표 만족도**: 4.0+/5.0
