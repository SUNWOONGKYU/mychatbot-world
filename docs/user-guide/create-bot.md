# 봇(페르소나) 생성 가이드

> 나만의 AI 페르소나를 처음부터 끝까지 만드는 단계별 안내

---

## 개요

MyChatbot World에서 봇(Bot) 또는 페르소나(Persona)는 나만의 AI 캐릭터입니다. 비즈니스 고객 응대, 개인 브랜딩, 교육 보조, 오락 등 원하는 목적에 맞게 설정할 수 있습니다. 한 번 만든 봇은 마켓플레이스에 등록하거나, QR 코드로 공유하거나, 웹사이트에 삽입할 수 있습니다.

---

## 1단계: 봇 생성 시작

1. 상단 내비게이션에서 **[봇 만들기]** 를 클릭합니다.
   또는 홈 대시보드의 **[+ 새 봇 만들기]** 버튼을 클릭합니다.

[스크린샷: 홈 대시보드 — 새 봇 만들기 버튼]

---

## 2단계: 기본 정보 입력

### 봇 이름

- 봇의 이름을 입력합니다. (최대 100자)
- 사용자에게 표시되는 이름이므로 기억하기 쉽고 목적을 잘 나타내는 이름을 권장합니다.
- 예: "친절한 고객센터 봇", "마케팅 전략가 Alex", "영어 회화 선생님 Jane"

[스크린샷: 봇 이름 입력 필드]

### 봇 설명

- 봇의 역할, 전문 분야, 성격을 자유롭게 설명합니다. (최대 2,000자)
- 설명이 구체적일수록 AI가 더 정확하게 페르소나를 분석하고 설정을 자동 추천합니다.
- 좋은 설명 예시:
  > "저는 10년 경력의 디지털 마케팅 전문가입니다. SEO, 콘텐츠 마케팅, SNS 전략에 대한 실무적인 조언을 드릴 수 있습니다. 친근하지만 전문적인 톤으로 대화합니다."

[스크린샷: 봇 설명 입력 폼]

> **팁:** 설명을 입력하면 AI가 자동으로 비즈니스 유형, 말투, 키워드를 분석하여 봇의 성격과 FAQ를 추천해 줍니다.

---

## 3단계: 성격 설정 — 감성 슬라이더

봇의 응답 스타일을 감성 슬라이더로 조절합니다.

[스크린샷: 감성 슬라이더 컴포넌트 — 1에서 100 범위]

### 슬라이더 값의 의미

| 범위 | 스타일 | 사용 예 |
|------|--------|---------|
| 1 ~ 30 | 정확하고 간결함 (Economy) | 고객 지원, 정보 제공 |
| 31 ~ 70 | 균형 잡힌 응답 (Standard) | 일반적인 비즈니스 대화 |
| 71 ~ 100 | 감성적이고 창의적 (Premium) | 엔터테인먼트, 스토리텔링 |

- **낮은 값 (Economy)**: 논리적이고 핵심만 전달하는 응답. 빠르고 크레딧 소모가 적습니다.
- **중간 값 (Standard)**: 정확성과 친근함을 균형 있게 유지하는 기본 설정.
- **높은 값 (Premium)**: 풍부한 표현, 감성적 언어, 창의적 답변. 크레딧 소모가 많습니다.

> **주의:** 슬라이더 값은 **기본값** 설정이며, 실제 대화 시 사용자가 이 값을 조정할 수 있습니다.

---

## 4단계: Knowledge Base (지식 저장소) 추가 (선택)

봇이 특정 주제나 데이터를 기반으로 답변하도록 지식을 추가합니다.

[스크린샷: Knowledge Base 업로드 폼]

### KB 추가 방법

1. 봇 생성 화면에서 **[지식 추가]** 탭을 클릭합니다.
2. 다음 중 하나를 선택합니다:
   - **텍스트 직접 입력**: 제목과 내용을 직접 작성
   - **파일 업로드**: PDF, TXT, Word 파일 업로드 (최대 10MB)
   - **URL**: 웹 페이지 URL 입력 (내용 자동 추출)
3. 업로드 후 **[임베딩 생성]** 버튼을 클릭합니다. (AI가 내용을 벡터로 변환)

> **팁:** KB를 추가하면 봇이 해당 내용을 참고해 더 정확한 답변을 제공합니다. 예를 들어 회사 FAQ 문서를 업로드하면 봇이 그 내용으로 고객에게 답변합니다.

---

## 5단계: 마켓플레이스 공개 설정

생성한 봇을 다른 사용자에게 공개할지 여부를 설정합니다.

[스크린샷: 마켓플레이스 공개 설정 토글]

### 공개 설정 옵션

| 설정 | 설명 |
|------|------|
| **비공개 (Private)** | 나만 사용 가능. 마켓플레이스에 표시되지 않음 |
| **공개 무료 (Public Free)** | 누구나 무료로 사용 가능 |
| **공개 유료 (Public Paid)** | 구독 또는 건당 결제 방식으로 수익 창출 |

유료 설정 시 추가로 입력할 항목:
- **가격 (월 구독)**: 0원 이상 (무료도 가능)
- **카테고리**: 비즈니스, 교육, 엔터테인먼트, 생산성, 기타
- **썸네일 이미지**: 마켓플레이스에 표시될 대표 이미지

---

## 6단계: 저장 및 테스트

1. 모든 설정을 완료한 후 **[봇 만들기]** 버튼을 클릭합니다.
2. AI가 자동으로:
   - 봇 분석 (비즈니스 유형, 톤, 키워드 추출)
   - FAQ 10개 자동 생성
   - 배포 URL 발급
   - QR 코드 생성
3. 완료 후 **테스트 채팅** 화면이 나타납니다.

[스크린샷: 봇 생성 완료 화면 — 배포 URL, QR 코드, 테스트 채팅]

### 테스트 방법

- 테스트 채팅창에서 봇과 직접 대화해 보세요.
- 의도한 대로 응답하는지 확인하고, 필요하면 봇 설명을 수정하여 다시 시도합니다.

---

## 봇 편집 방법

생성 후 언제든지 봇 설정을 변경할 수 있습니다.

1. **MyPage > 내 봇 목록**으로 이동합니다.
2. 수정할 봇 카드의 **[설정]** 또는 **[편집]** 버튼을 클릭합니다.
3. 다음 항목을 수정할 수 있습니다:
   - 봇 이름, 설명
   - 감성 슬라이더 기본값
   - Knowledge Base (추가/삭제)
   - 마켓플레이스 공개 여부 및 가격
   - 고급 설정: AI 모델, 온도(Temperature), 최대 토큰 수, 기본 언어

[스크린샷: 봇 편집 화면]

---

## 마켓플레이스에 등록하는 방법

봇 생성 후 마켓플레이스 등록을 따로 진행하려면:

1. MyPage > 내 봇 목록에서 해당 봇을 선택합니다.
2. **[마켓플레이스 등록]** 버튼을 클릭합니다.
3. 아래 정보를 입력합니다:
   - 공개 제목 (마켓플레이스 표시용)
   - 봇 소개 문구 (최대 500자)
   - 카테고리
   - 썸네일 이미지
   - 유료/무료 여부 및 가격
4. **[등록 신청]** 버튼 클릭

[스크린샷: 마켓플레이스 등록 폼]

> **주의:** 신규 등록 봇은 플랫폼 검토 과정을 거칩니다. 검토는 최대 24시간이 소요됩니다.

## 유의사항

### 저작권

- 타인의 저작물(글, 이미지, 영상 등)을 무단으로 Knowledge Base에 업로드하지 마세요.
- 봇 이름이나 설명에 실존 인물이나 브랜드를 사칭하면 서비스 이용이 제한될 수 있습니다.

### 콘텐츠 정책

다음의 봇 생성은 금지되어 있으며, 발견 시 즉시 삭제되고 계정이 제재를 받을 수 있습니다:
- 불법적인 정보(마약, 무기, 해킹 등) 제공 봇
- 성인 또는 혐오 콘텐츠 봇
- 타인을 사칭하거나 허위 정보를 유포하는 봇
- 보이스피싱 등 사기 목적의 봇

콘텐츠 정책 전문은 서비스 이용약관에서 확인할 수 있습니다.

## FAQ

**Q. 봇 설명을 어떻게 쓰면 더 좋은 봇이 만들어지나요?**
> 봇의 역할, 전문 분야, 대화 스타일, 금지 주제를 구체적으로 서술하세요. "전문적이지만 친근하게", "항상 질문으로 마무리" 같은 행동 지침도 도움이 됩니다.

**Q. 봇이 엉뚱한 답변을 할 때는 어떻게 하나요?**
> 봇 설명(프롬프트)을 더 구체적으로 수정하거나, Knowledge Base에 관련 내용을 추가하세요. 특정 주제에서 벗어나지 않도록 "다른 주제에 대한 질문은 정중히 거절하세요"와 같은 제약 문구를 추가하는 것도 효과적입니다.

**Q. QR 코드는 어디에 사용하나요?**
> 생성된 QR 코드를 명함, 포스터, 웹사이트, SNS에 삽입하면 스캔만으로 봇과 대화가 시작됩니다. 오프라인 홍보에도 활용할 수 있습니다.

**Q. 봇을 내 웹사이트에 삽입할 수 있나요?**
> 각 봇에 발급된 배포 URL을 iframe으로 삽입할 수 있습니다. 상세 방법은 MyPage > 내 봇 > [삽입 코드 보기]에서 확인하세요.
