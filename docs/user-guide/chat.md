# 대화 사용 가이드

> AI 페르소나와 대화하는 모든 기능의 사용 방법

---

## 개요

MyChatbot World의 대화 기능은 단순한 Q&A를 넘어, 감성 슬라이더로 AI의 응답 스타일을 실시간으로 조절하고, 대화 이력을 저장하며, 원하는 페르소나를 구독하여 반복 이용할 수 있는 종합 대화 경험을 제공합니다.

---

## 1. 대화 시작 방법

### 방법 1: 내 봇과 대화

1. 상단 메뉴 **[홈]** 또는 **[내 봇]** 탭을 클릭합니다.
2. 대화하고 싶은 봇 카드를 클릭합니다.
3. 하단 입력창에 메시지를 입력하고 **[보내기]** 버튼을 클릭하거나 `Enter`를 누릅니다.

[스크린샷: 대화 화면 — 입력창, 전송 버튼]

### 방법 2: 마켓플레이스에서 봇 찾아 대화

1. 상단 메뉴 **[마켓플레이스]** 를 클릭합니다.
2. 카테고리 또는 검색으로 원하는 봇을 찾습니다.
3. 봇 상세 페이지에서 **[대화하기]** 버튼을 클릭합니다.
4. 처음 이용하는 봇이라면 **[구독하기]** 또는 **[무료로 시작]** 버튼이 표시됩니다.

[스크린샷: 마켓플레이스 봇 상세 페이지]

### 방법 3: QR 코드 / URL로 접속

오프라인에서 QR 코드를 스캔하거나 직접 배포 URL로 접속하면 즉시 대화 화면이 열립니다.
- 로그인하지 않아도 게스트 모드로 대화할 수 있습니다.
- 게스트 모드에서는 대화 이력이 저장되지 않으며, 일부 기능이 제한됩니다.

## 2. 감성 슬라이더 사용법

감성 슬라이더는 대화 중 AI 응답의 **감성·창의성 수준**을 실시간으로 조절합니다.

[스크린샷: 대화 화면 오른쪽 패널 — 감성 슬라이더]

### 슬라이더 조작 방법

- 대화 화면의 **우측 사이드바** 또는 **하단 컨트롤 패널**에서 슬라이더를 찾습니다.
- 드래그하거나 화살표 버튼으로 값을 조정합니다. (1 ~ 100)
- 슬라이더를 조정하면 다음 메시지부터 새 설정이 적용됩니다.

### 슬라이더 값 가이드

| 값 범위 | 레벨 | AI 스타일 | 크레딧 소모 |
|---------|------|-----------|-----------|
| 1 ~ 33 | Economy | 짧고 정확한 답변, 논리 중심 | 적음 |
| 34 ~ 66 | Standard | 균형 잡힌 답변, 친근한 어조 | 보통 |
| 67 ~ 100 | Premium | 감성적, 창의적, 풍부한 표현 | 많음 |

> **팁:** 정보 조회나 업무 목적이라면 낮은 값(Economy)이, 대화나 창작 목적이라면 높은 값(Premium)이 더 적합합니다.

---

## 3. 비용 슬라이더 사용법

비용 슬라이더는 **사용할 AI 모델의 품질 티어**를 선택합니다.

[스크린샷: 비용 슬라이더]

- **Economy**: 가벼운 모델, 빠르고 경제적
- **Standard**: 기본 모델, 품질과 비용 균형
- **Premium**: 고급 모델, 최고 품질

> **주의:** 비용 슬라이더 값이 높을수록 응답 품질이 좋아지지만 크레딧 소모도 증가합니다. 각 대화 시작 전 예상 크레딧 소모량이 표시됩니다.

---

## 4. 음성 기능 사용 (TTS / STT)

### 텍스트 음성 변환 (TTS: Text-to-Speech)

AI의 응답을 음성으로 들을 수 있습니다.
1. 대화 화면 설정에서 **[음성 응답]** 을 활성화합니다.
2. AI가 답변할 때 자동으로 음성이 재생됩니다.
3. 개별 메시지의 스피커 아이콘을 클릭해 특정 메시지만 재생할 수도 있습니다.

[스크린샷: 메시지 우측 스피커 아이콘]

### 음성 인식 (STT: Speech-to-Text)

마이크로 메시지를 입력할 수 있습니다.
1. 입력창 옆 **마이크 아이콘**을 클릭합니다.
2. 브라우저에서 마이크 사용 권한을 허용합니다.
3. 말하면 텍스트로 자동 변환됩니다.
4. 변환된 텍스트를 확인하고 **[보내기]** 를 클릭합니다.

[스크린샷: 마이크 버튼 활성화 상태]

## 5. 대화 이력 관리

### 대화 이력 저장

로그인 상태에서의 모든 대화는 자동으로 저장됩니다.

- **MyPage > 대화 이력** 탭에서 이전 대화를 불러올 수 있습니다.
- 대화 목록에서 특정 봇과의 대화를 클릭하면 전체 내용이 표시됩니다.

[스크린샷: 대화 이력 목록]

### 대화 이어서 하기

이전 대화를 이어서 진행하려면:
1. 대화 이력 목록에서 이어서 할 대화를 클릭합니다.
2. 해당 대화 화면으로 이동합니다.
3. 동일한 `conversationId`가 유지되어 맥락이 이어집니다.

### 대화 이력 삭제

1. 대화 이력 목록에서 삭제할 대화를 선택합니다.
2. 우측 **[삭제]** 버튼 또는 `[...]` 메뉴에서 삭제를 클릭합니다.
3. 확인 팝업에서 **[삭제 확인]** 을 클릭합니다.

> **주의:** 삭제된 대화 이력은 복구할 수 없습니다.

## 6. 페르소나 구독 및 사용 방법

### 구독이란?

마켓플레이스의 유료 봇을 정기적으로 이용하려면 구독이 필요합니다.

[스크린샷: 봇 구독 화면]

### 구독 방법

1. 마켓플레이스에서 원하는 봇의 상세 페이지를 엽니다.
2. **[구독하기]** 버튼을 클릭합니다.
3. 구독 가격과 결제 방식을 확인합니다.
4. 크레딧 잔액이 충분하면 즉시 구독이 활성화됩니다.

### 구독 관리

- **MyPage > 구독 관리**에서 현재 구독 중인 봇 목록을 확인할 수 있습니다.
- 구독 해지: 구독 목록에서 해지할 봇의 **[구독 해지]** 버튼 클릭

## 7. 대화 품질 팁

더 나은 AI 응답을 얻기 위한 실용적인 팁입니다.

### 질문을 구체적으로 작성하세요

```
나쁜 예: "마케팅 알려줘"
좋은 예: "소규모 카페를 위한 인스타그램 마케팅 전략 3가지를 알려줘. 예산은 월 50만원이야."
```

### 배경 정보를 제공하세요

```
나쁜 예: "이메일 써줘"
좋은 예: "신규 고객에게 첫 구매 감사 인사와 10% 할인 쿠폰을 안내하는 이메일을 작성해줘. 우리 브랜드는 친환경 화장품 브랜드야."
```

### 감성 슬라이더를 목적에 맞게 설정하세요

- 사실 정보 검색, 업무 지원 → 슬라이더를 낮게 (Economy)
- 창작, 스토리텔링, 감성 대화 → 슬라이더를 높게 (Premium)

### 대화를 이어가며 다듬으세요

AI의 첫 응답이 완벽하지 않아도 괜찮습니다. "더 짧게", "구체적인 예시 추가", "다른 방식으로 다시 써줘" 같은 후속 메시지로 원하는 결과를 만들어 나갈 수 있습니다.

## FAQ

**Q. 대화 중 갑자기 연결이 끊겼어요. 이전 대화를 복구할 수 있나요?**
> 로그인 상태였다면 대화 이력에서 마지막 저장 시점까지 복구할 수 있습니다. 하지만 전송 도중 끊긴 메시지는 복구되지 않을 수 있습니다.

**Q. 봇이 이전 대화 내용을 기억하지 못해요.**
> 대화 맥락은 `conversationId`를 기반으로 유지됩니다. 새 대화를 시작하면 맥락이 초기화됩니다. 이전 대화를 이어가려면 대화 이력에서 해당 대화를 선택해 접속하세요.

**Q. 게스트 모드와 로그인 모드의 차이는 무엇인가요?**
> 게스트 모드에서는 대화가 저장되지 않고, 유료 봇 접근, 구독 기능, 대화 이력 확인 등이 제한됩니다. 더 많은 기능을 이용하려면 로그인하세요.

**Q. 대화에서 부적절한 응답이 나왔어요. 어떻게 하나요?**
> 메시지 우측 **[신고]** 버튼(깃발 아이콘)을 클릭하여 부적절한 응답을 신고할 수 있습니다. 신고된 내용은 검토 후 봇 개선에 반영됩니다.

## 트러블슈팅

### AI 응답이 느릴 때

1. **감성/비용 슬라이더 값 낮추기**: 고급 모델일수록 응답 생성에 더 오래 걸립니다.
2. **네트워크 상태 확인**: 안정적인 인터넷 연결인지 확인하세요.
3. **스트리밍 방해 금지**: 응답 생성 중에는 페이지를 새로고침하지 마세요. 텍스트가 스트리밍 방식으로 출력되는 것은 정상입니다.

### 음성 기능이 작동하지 않을 때

1. **브라우저 권한 확인**: 주소창 왼쪽의 자물쇠 아이콘을 클릭하여 마이크 권한을 허용했는지 확인하세요.
2. **지원 브라우저 확인**: Chrome, Edge, Firefox 최신 버전을 권장합니다. Safari에서는 일부 기능이 제한될 수 있습니다.
3. **마이크 연결 확인**: 컴퓨터의 마이크가 정상적으로 작동하는지 운영체제 설정에서 확인하세요.

### 크레딧이 예상보다 많이 소모될 때

1. **슬라이더 값 확인**: Premium 슬라이더는 고급 모델을 사용하여 크레딧 소모가 많습니다.
2. **긴 대화 이력**: 대화가 길어질수록 컨텍스트 처리에 더 많은 토큰이 사용됩니다. 새 대화를 시작하면 크레딧 효율이 높아집니다.
3. **크레딧 이력 확인**: MyPage > 크레딧 이력에서 상세 차감 내역을 확인할 수 있습니다.
