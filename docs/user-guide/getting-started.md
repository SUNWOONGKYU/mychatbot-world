# Getting Started - 사용자 가이드

AI Avatar Chat Platform 시작 가이드입니다.

---

## 🚀 빠른 시작

### 1. 회원가입

1. 웹사이트 접속: https://example.com
2. "Sign Up" 클릭
3. 이메일, 사용자명, 비밀번호 입력
4. "Create Account" 클릭

### 2. 페르소나 선택

6가지 AI 챗봇 중 선택:
- 💼 **비즈니스 어시스턴트**: 일정, 이메일, 데이터 분석
- 🎧 **고객 서비스**: FAQ, 문제 해결
- 📚 **교육 튜터**: 학습 지원, 퀴즈
- ⚕️ **헬스케어**: 건강 정보, 웰니스 팁
- 🎭 **엔터테인먼트**: 스토리, 게임, 추천
- 🤖 **개인 비서**: 일상 지원

### 3. 대화 시작

1. 메시지 입력창에 질문 입력
2. "Send" 버튼 클릭 (또는 Enter)
3. AI 응답 및 3D 아바타 감정 표현 확인

---

## 💡 주요 기능

### 음성 입력 (선택)
- 🎤 버튼 클릭
- 음성으로 메시지 말하기
- 자동으로 텍스트 변환

### 대화 관리
- 과거 대화 목록 확인
- 검색으로 대화 찾기
- 불필요한 대화 삭제

### 아바타 커스터마이징
- 아바타 색상 변경
- (향후) 의상, 표정 커스터마이징
