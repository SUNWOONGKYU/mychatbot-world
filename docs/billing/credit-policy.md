# MyChatbot World 크레딧 과금 정책

> 최초 작성: 2026-04-12
> 적용 대상: 모든 대화 및 프리미엄 기능 활성화

---

## 1. 크레딧 기본 단위

- **1 크레딧 = 1원 (₩)**
- 모든 대화 비용은 실제 사용한 AI 토큰 수로 계산 (모델 무관)
- 구독 요금제·초과·단건 충전 모두 동일 단위 사용

---

## 2. 원가 계산 공식

```
원가(₩) = ( 입력토큰 × 모델_입력단가($/M) / 1,000,000
           + 출력토큰 × 모델_출력단가($/M) / 1,000,000 ) × 환율(₩/$)
```

- 모델 단가: 사용된 AI 모델의 실제 API 가격 적용
- 환율: 서비스 설정값 기준 (기본 $1 = ₩1,380)
- 입력/출력 토큰: 매 API 응답에서 실제 사용량 취득

---

## 3. 크레딧 차감 규칙

| 구분 | 크레딧 차감 | 원가 대비 |
|------|-----------|---------|
| **구독 요금제 내** | `CEIL(원가 × 2)` | 원가의 2배 |
| **요금제 초과 시** | `CEIL(원가 × 4)` | 원가의 4배 |
| **단건 충전 크레딧** | `CEIL(원가 × 4)` | 원가의 4배 (구독 잔액 유무와 무관) |

- 구독 크레딧 소진 → 자동으로 초과 요금(×4) 적용
- 잔액 부족 시 대화 불가 (크레딧 충전 안내)

> **구현 방식:** `mcw_credits.subscription_balance > 0`이면 ×2, `0`이면 ×4를 적용한다.
> 구독 요금제 크레딧 및 환영 크레딧은 `subscription_balance`에 포함되어 지급된다.
> 단건 충전 크레딧은 `subscription_balance`에 포함되지 않으므로 자동으로 ×4가 적용된다.

---

## 4. 구독 요금제 (정기 크레딧 패키지)

| 단계 | 월 요금(VAT 포함) | 제공 크레딧 | 소비 가능 원가(₩) |
|------|---------------|-----------|----------------|
| 일단 (Dan I)   | ₩9,900  | 9,000크레딧  | ₩4,500  |
| 이단 (Dan II)  | ₩29,700 | 27,000크레딧 | ₩13,500 |
| 삼단 (Dan III) | ₩99,000 | 90,000크레딧 | ₩45,000 |
| 사단 (Dan IV)  | ₩297,000 | 270,000크레딧 | ₩135,000 |

> **산출 근거:** 월 요금 ÷ 1.1(부가세 제거) = 크레딧 수
> (예: ₩9,900 ÷ 1.1 = 9,000크레딧)
> 크레딧을 구독 요금(×2)으로 소비하면 원가 기준 1/2 가치

## 5. 초과 요금 비교

| 크레딧 | 구독 내 소비 가능 원가 | 초과 시 소비 가능 원가 |
|------|------------------|------------------|
| 9,000크레딧 | ₩4,500 | ₩2,250 |
| 27,000크레딧 | ₩13,500 | ₩6,750 |
| 90,000크레딧 | ₩45,000 | ₩22,500 |
| 270,000크레딧 | ₩135,000 | ₩67,500 |

→ 초과 시 동일 크레딧으로 구독 내 대비 절반만 사용 가능

## 6. 단건 충전 (애드온)

| 금액 | 크레딧 |
|------|------|
| ₩10,000  | 10,000크레딧  |
| ₩30,000  | 30,000크레딧  |
| ₩50,000  | 50,000크레딧  |
| ₩100,000 | 100,000크레딧 |

- 최소 충전 금액: ₩10,000
- 단건 충전 크레딧은 유효기간 없음
- 단건 크레딧 소비 시 요금 구분: 구독 잔액 유무와 무관하게 초과 요금(×4) 기준 적용
