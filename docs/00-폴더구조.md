# 폴더 구조 설명

## 📂 각 폴더의 용도

### `/docs` - 문서
- 프로젝트 계획서
- API 문서
- 사용자 가이드
- 개발 가이드

### `/src` - 소스 코드
- 백엔드 코드
- 프론트엔드 코드
- 공통 라이브러리

### `/assets` - 디자인 에셋
- 이미지
- 아이콘
- 폰트
- UI 컴포넌트

### `/tests` - 테스트
- 단위 테스트
- 통합 테스트
- E2E 테스트

### `/config` - 설정
- 환경 변수
- AI 모델 설정
- 데이터베이스 설정

### `/scripts` - 스크립트
- 빌드 스크립트
- 배포 스크립트
- 유틸리티 도구

### `/agents` - AI 에이전트
- 서브에이전트 코드
- 에이전트 설정
- 프롬프트 템플릿

### `/avatars` - 아바타
- 3D 모델 파일
- 애니메이션 데이터
- 텍스처

### `/data` - 데이터
- 훈련 데이터
- 샘플 데이터
- 데이터베이스 스키마

### `/logs` - 로그
- 애플리케이션 로그
- 에러 로그
- 성능 로그
