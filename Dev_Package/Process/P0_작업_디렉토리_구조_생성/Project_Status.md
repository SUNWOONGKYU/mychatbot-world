## 📋 프로젝트 기본 정보

### 프로젝트 개요
- **목표**: 음성 3분으로 AI 챗봇을 생성하고, 스킬 마켓을 통해 확장하며, 멀티채널로 배포하는 SaaS 플랫폼
- **핵심 차별점**:
  - 🎤 음성 3분으로 AI 챗봇 생성
  - 🏪 스킬 마켓플레이스 (21개 스킬)
  - 📱 QR 코드 + 멀티채널 배포
  - 🎯 5개 분야별 맞춤 템플릿 (정치인, 유튜버, 기업가, 강사, 식당)

### 기술 스택
- **프론트엔드**: Vanilla HTML/CSS/JavaScript
- **백엔드**: Vercel Serverless Functions
- **AI**: OpenRouter API (GPT-4o, Claude, Gemini)
- **음성**: Web Speech API + OpenAI Whisper
- **DB**: Supabase (PostgreSQL)
- **배포**: Vercel
