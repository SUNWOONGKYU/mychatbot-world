# My Chatbot World - 프로토타입 계획

## 핵심 화면 4개
1. **랜딩 페이지** (index.html) - 서비스 소개, CTA
2. **챗봇 생성** (create.html) - 템플릿 선택, 음성/텍스트 입력, AI 분석
3. **채팅 인터페이스** (bot.html) - 실시간 대화, FAQ
4. **대시보드** (dashboard.html) - 통계, FAQ 관리, 스킬

## 기술 스택
- Vanilla HTML/CSS/JS (빠른 프로토타이핑)
- Vercel Serverless Functions (API)
- OpenRouter API (AI)
- Web Speech API (음성)
- localStorage (MVP 데이터)

## 프로토타입 범위
- ✅ 전체 UI 플로우 (4개 페이지 간 이동)
- ✅ 음성 녹음 → AI 분석 → 챗봇 생성
- ✅ 실시간 채팅 (LLM 연동)
- ✅ 대시보드 기본 기능
- ❌ 결제 연동 (MVP 이후)
- ❌ 사용자 인증 (MVP 이후)
