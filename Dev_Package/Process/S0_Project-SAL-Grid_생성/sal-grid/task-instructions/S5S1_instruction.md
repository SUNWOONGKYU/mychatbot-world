# S5S1: 개인정보처리방침 페이지 + 동의 플로우

## Task 정보
- **Task ID**: S5S1
- **Task Name**: 개인정보처리방침 페이지 + 동의 플로우
- **Stage**: S5 (품질 개선)
- **Area**: S (Security)
- **Dependencies**: S4S1

## Task 목표

GDPR 및 개인정보보호법(PIPA) 준수를 위해 개인정보처리방침 페이지를 구현하고, 회원가입 시 명시적 동의 체크박스 플로우를 추가한다.

## 구현 범위

### 1. 개인정보처리방침 페이지
- 경로: `/privacy` 또는 `/privacy-policy`
- 내용: 수집 항목, 이용 목적, 보유 기간, 제3자 제공, 파기 절차
- 최종 업데이트 날짜 표시

### 2. 이용약관 페이지
- 경로: `/terms`
- 서비스 이용 약관 내용

### 3. 회원가입 동의 플로우
- 필수 동의: 이용약관, 개인정보처리방침
- 선택 동의: 마케팅 정보 수신
- 체크박스 미체크 시 가입 불가 처리

## 생성/수정 파일

| 파일 | 변경 내용 |
|------|----------|
| `app/privacy/page.tsx` | 개인정보처리방침 페이지 신규 생성 |
| `app/terms/page.tsx` | 이용약관 페이지 신규 생성 |
| `app/auth/signup/page.tsx` 또는 관련 컴포넌트 | 동의 체크박스 추가 |
| `app/api/auth/signup/route.ts` 또는 관련 API | 동의 여부 검증 로직 추가 |

## 완료 기준

- [ ] 개인정보처리방침 페이지 접근 가능 (/privacy)
- [ ] 이용약관 페이지 접근 가능 (/terms)
- [ ] 회원가입 화면에 필수 동의 체크박스 존재
- [ ] 미동의 상태에서 가입 시도 시 오류 처리
- [ ] 동의 여부가 DB에 저장됨 (또는 서버에서 검증됨)
