# S3BA7: Community(봇마당) API

## Task 정보
- **Task ID**: S3BA7
- **Task Name**: Community(봇마당) API
- **Stage**: S3 (개발 2차)
- **Area**: BA (Backend APIs)
- **Dependencies**: S3DB3

## Task 목표
봇마당 커뮤니티 API. 게시글 CRUD, 댓글 CRUD, 좋아요 토글, 신고, 카테고리 관리.

## 생성/수정 파일
| 파일 | 변경 내용 |
|------|----------|
| `api/Backend_APIs/community-post.js` | 게시글 CRUD API - 작성, 수정, 삭제, 조회 |
| `api/Backend_APIs/community-comment.js` | 댓글 CRUD API - 댓글 관리 |
| `api/Backend_APIs/community-like.js` | 좋아요 토글 API |
| `api/Backend_APIs/community-report.js` | 신고 API - 부적절한 콘텐츠 신고 |
| `api/Backend_APIs/community-category.js` | 카테고리 관리 API |
