## 배경 및 목적

신규 사용자가 회원가입 없이 AI 챗봇의 핵심 기능을 먼저 체험할 수 있도록 게스트 모드를 제공한다. 체험 제한(봇 1개 생성, 메시지 10개)을 통해 서비스 남용을 방지하면서도 전환율을 높인다.

## 세부 작업 지시

1. pages/guest/ 폴더에 게스트 체험 페이지 생성:
   - index.html: 게스트 체험 시작 페이지 (직업 선택 → 봇 미리보기)
   - chat.html: 게스트 대화 페이지 (사용량 제한 UI 포함)
2. js/guest-mode.js 작성:
   - 게스트 세션 관리 (localStorage 기반)
   - 봇 생성 1회 제한 체크
   - 메시지 10개 제한 카운터
   - 제한 도달 시 회원가입 유도 모달
3. 랜딩 페이지(S2F1)에 "지금 무료로 체험하기" 버튼 추가
4. S2BA4 게스트 API와 연동
