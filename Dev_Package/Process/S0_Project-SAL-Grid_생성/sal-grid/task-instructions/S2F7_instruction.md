## 배경 및 목적

챗봇 생성 위저드(S2F4)의 Step 4 (인사말 설정)에서 사용자가 설정한 인사말이 실제로 어떻게 들리는지 미리 확인할 수 있는 TTS 시청 버튼을 추가한다. 이를 통해 사용자가 챗봇 음성을 확인하고 조정할 수 있다.

## 세부 작업 지시

1. pages/create/index.html의 Step 4 섹션에 "인사말 듣기" 버튼 추가
2. 버튼 클릭 시 /api/tts 엔드포인트 호출:
   - 현재 입력된 인사말 텍스트 전달
   - 선택된 음성 프로파일 전달
3. TTS 응답(audio blob) 수신 후 브라우저 Audio API로 재생
4. 재생 중 상태 표시 (로딩 스피너, 재생 중 아이콘)
5. 재생 완료/오류 시 버튼 상태 초기화
