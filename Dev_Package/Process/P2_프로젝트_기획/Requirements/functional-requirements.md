# My Chatbot World - 기능 요구사항

## 1. 랜딩 페이지
- FR-001: 핵심 가치 제안 표시 ("음성 3분으로 AI 챗봇 생성")
- FR-002: 5개 분야별 템플릿 미리보기
- FR-003: 가격표 표시 (Free/Basic/Pro/Enterprise)
- FR-004: CTA 버튼 → 챗봇 생성 페이지 이동
- FR-005: 모바일 반응형 레이아웃

## 2. 챗봇 생성 플로우
- FR-010: 분야 선택 (5개 템플릿)
- FR-011: 기본 정보 입력 (이름, 설명)
- FR-012: 음성 입력 (Web Speech API, 3분 타이머)
- FR-013: 텍스트 입력 (대안)
- FR-014: AI 분석 → 챗봇 성격/어조/인사말/FAQ 자동 생성
- FR-015: 미리보기 및 수정
- FR-016: 생성 완료 → URL + QR 코드 제공

## 3. 채팅 인터페이스
- FR-020: 고유 URL로 챗봇 접속 (/bot/{username})
- FR-021: 실시간 채팅 UI (메시지 버블, 타이핑 인디케이터)
- FR-022: FAQ 빠른 답변 버튼
- FR-023: 음성 입력/출력 지원
- FR-024: 모바일 최적화

## 4. 대시보드
- FR-030: 챗봇 목록 및 관리
- FR-031: 대화 통계 (일별/주별/월별)
- FR-032: FAQ 관리 (추가/수정/삭제)
- FR-033: 스킬 마켓플레이스 (설치/제거)
- FR-034: 배포 관리 (QR 코드, 임베드 코드, 멀티채널)
- FR-035: 디자인 커스터마이징

## 5. API
- FR-040: 챗봇 생성 API (음성 분석 + 성격 추출)
- FR-041: 대화 API (LLM 응답, 컨텍스트 유지)
- FR-042: TTS API (텍스트 → 음성)
