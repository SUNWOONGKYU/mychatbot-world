## 1. 임무 요약

**재도전성공패키지 사업계획서를 아래한글(HWP) 파일로 변환하라.**

현재 Word(.docx) 파일이 생성되어 있으나, 정부지원사업 제출 양식이 HWP이므로
한글이 설치된 PC에서 HWP 파일로 변환해야 한다.

## 4. 작업 방법 (2가지 중 택 1)

### 방법 A: HWP-MCP (권장 — Claude가 한글 직접 조작)

**전제조건**: 한글(HWP) 2010 이상 설치, Python 3.7+, Windows

```bash
# 1. HWP-MCP 설치
pip install hwp-mcp
# 또는
git clone https://github.com/jkf87/hwp-mcp.git
cd hwp-mcp
pip install -r requirements.txt

# 2. Claude Code MCP 설정 (settings.json에 추가)
# ~/.claude/settings.json → mcpServers에 hwp-mcp 등록
```

**작업 흐름**:
1. `별첨1. 2026년 재도전성공패키지 사업계획서 양식.hwp`를 한글로 열기
2. MD 원본(`사업계획서_재도전성공패키지_MyChabbotWorld.md`)을 읽기
3. 양식의 각 섹션에 맞춰 내용 삽입
4. 표는 한글의 표 기능으로 생성
5. 저장

### 방법 B: 한글 COM 자동화 (HWP-MCP 없이 직접 제어)

```python
# 한글 COM 인터페이스 직접 사용
import win32com.client

hwp = win32com.client.Dispatch("HWPFrame.HwpObject")
hwp.RegisterModule("FilePathCheckDLL", "FilePathCheckerModule")
hwp.Open("별첨1. 양식.hwp")

# 텍스트 삽입
hwp.HAction.GetDefault("InsertText", hwp.HParameterSet.HInsertText.HSet)
hwp.HParameterSet.HInsertText.Text = "삽입할 텍스트"
hwp.HAction.Execute("InsertText", hwp.HParameterSet.HInsertText.HSet)

# 저장
hwp.SaveAs("사업계획서_완성.hwp")
hwp.Quit()
```

### 방법 C: 가장 간단한 방법 (수동 + AI 보조)

1. 한글에서 `별첨1. 사업계획서 양식.hwp` 열기
2. DOCX 파일(`사업계획서_재도전성공패키지_MyChatbotWorld.docx`)을 한글에서 열기 (한글은 DOCX 읽기 지원)
3. DOCX에서 내용을 복사하여 양식에 붙여넣기
4. 서식 정리 후 저장

## 5. 서식 스펙 (generate_docx.py 기준)

| 항목 | 값 |

| 기본 폰트 | Malgun Gothic (맑은 고딕) |
| 본문 크기 | 10pt |

| 중제목 (H2) | 12pt Bold |

| 표 헤더 | 배경색 #D6E4F0 (연한 파랑), 가운데 정렬, Bold |
| 표 본문 | 9pt, 좌측 정렬 |

## 6. 주의사항

1. **양식 우선**: `별첨1. 사업계획서 양식.hwp`의 구조와 섹션 순서를 반드시 따를 것
2. **내용 원본**: MD 파일이 최종 버전. DOCX는 MD에서 자동 생성된 것
3. **표 서식**: 정부 양식에 맞게 깔끔하게. 헤더 행 배경색 적용
4. **페이지 구분**: 대단원(# 1, # 2, # 3, # 4) 시작 시 페이지 나누기
5. **아스키 아트 다이어그램** (2-2절): 고정폭 폰트(Consolas 또는 D2Coding)로 삽입
6. **각주**: 비용 슬라이더 설명 등 `*¹` 표시된 부분은 각주로 처리

## 7. 검증 체크리스트

작업 완료 후 확인할 사항:

- [ ] 양식(별첨1)의 모든 섹션이 빠짐없이 채워졌는가

- [ ] 표 헤더에 배경색이 적용되었는가
- [ ] 페이지 나누기가 대단원별로 적용되었는가
- [ ] 폰트가 맑은 고딕으로 통일되었는가
- [ ] 아스키 아트 다이어그램이 고정폭 폰트로 삽입되었는가
- [ ] 파일명: `사업계획서_재도전성공패키지_MyChatbotWorld.hwp`로 저장
