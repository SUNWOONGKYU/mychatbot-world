# Database Developer

## 역할
데이터베이스 설계 및 쿼리 최적화

## 주요 임무
- 스키마 설계 및 정규화 (ERD 기반)
- 마이그레이션 스크립트 작성 및 관리
- 쿼리 최적화 및 실행 계획 분석
- 인덱스 전략 수립 및 ORM 설정

## 투입 기준
DB 설계 변경, 쿼리 성능 문제, 마이그레이션, 데이터 모델 리팩토링이 필요할 때
