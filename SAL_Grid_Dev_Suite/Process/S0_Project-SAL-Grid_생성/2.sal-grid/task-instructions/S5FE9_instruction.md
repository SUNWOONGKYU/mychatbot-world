# S5FE9: 게스트 모드 리디자인

## Task 정보
- **Task ID**: S5FE9
- **Task Name**: 게스트 모드 리디자인
- **Stage**: S5 (디자인 혁신)
- **Area**: FE (Frontend)
- **Dependencies**: S5DS4, S5FE1, S5FE2

## Task 목표

비로그인 상태의 게스트 모드 UI를 S5FE1 디자인 시스템으로 전면 리디자인한다. 게스트가 챗봇과 대화하는 공개 채팅 페이지(/chat/[botId])와 게스트 전용 UI(로그인 유도 배너, 제한 안내)를 포함한다.

## 구현 항목

1. **게스트 채팅 페이지 (/chat/[botId])**:
   - 비로그인 상태에서도 접근 가능한 공개 채팅 UI
   - 봇 프로필 헤더 (아바타, 이름, 설명)
   - 채팅 메시지 버블 (새 디자인 시스템 적용)
   - 로그인 유도 배너 (N회 대화 후 표시)
   - TTS 버튼 (게스트도 사용 가능)

2. **게스트 랜딩 요소**:
   - 게스트 모드 상태바 (상단 - "로그인하면 내 챗봇을 만들 수 있어요")
   - 회원가입 CTA (앰버 강조, 슬라이드업 애니메이션)
   - 봇 소개 카드 (봇 이름, 스킬, 운영자 정보)

3. **공유 링크 랜딩**:
   - 배포 URL 접근 시 봇 소개 + 채팅 시작 화면
   - OG 태그 최적화 (봇 이름/아바타 공유 시 미리보기)
