1. **랜딩 페이지 (/)**: Hero 섹션, 기능 소개, 스킬 미리보기, CTA 구성
2. **Birth (챗봇 생성)**: 위저드 3단계 레이아웃 (목적 → 스킬 → 배포)
3. **5대 메뉴**: Learning / Skills / Jobs / Community 각 페이지 레이아웃
4. **대시보드 (/dashboard)**: 봇 카드 목록, 수익 위젯, 빠른 실행 버튼
5. **마이페이지 (/mypage)**: 프로필, 수익 현황, 구독 관리
6. **마켓플레이스 + 비즈니스**: 스킬 목록/상세, 수익 대시보드
