## 원칙 예시 (확정 전 초안)

1. **Clarity over cleverness** — 혼란보다는 명료
2. **Content first, chrome second** — 콘텐츠가 주인공
3. **Motion with purpose** — 장식이 아닌 안내로서의 모션
4. **Dark-first, Light-equal** — 다크를 우선 설계, 라이트는 동등 품질
5. **Dense but breathable** — 정보 밀도와 여백의 균형
6. **Accessible by default** — 접근성은 옵션이 아님
7. **Korean-friendly typography** — 한글 가독성 최적화
