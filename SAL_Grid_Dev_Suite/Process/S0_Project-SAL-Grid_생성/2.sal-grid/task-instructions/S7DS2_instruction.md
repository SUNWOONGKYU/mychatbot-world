| Vercel | 다크모드·스페이싱 | 코드 블록 스타일, 그리드 시스템, elevation |

| Arc | 인터랙션 밀도 | 사이드바/커맨드바, 컨텍스트 메뉴 |
| Raycast | 컴포넌트 정교함 | 모달/팔레트, 단축키 표시, Toast |
