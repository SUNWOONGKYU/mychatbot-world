# S4F10: File System Access API 로컬 폴더 연결 - 검증 지침

## 검증 정보
- **Task ID**: S4F10
- **Verification Agent**: code-reviewer

## 검증 항목

### 1. API 기능 검증
- [ ] Chrome/Edge에서 "폴더 연결" 클릭 시 OS 폴더 선택 대화상자가 뜨는가?
- [ ] 폴더 선택 후 파일 읽기가 정상 작동하는가?
- [ ] 폴더 선택 후 파일 쓰기가 정상 작동하는가?
- [ ] 하위 폴더 탐색이 정상 작동하는가?

### 2. 핸들 저장/복원 검증
- [ ] IndexedDB에 디렉토리 핸들이 정상 저장되는가?
- [ ] 페이지 새로고침 후 핸들 복원이 되는가?
- [ ] 복원 시 권한 재확인 프롬프트가 뜨는가?
- [ ] 권한 승인 후 기존 폴더에 다시 접근 가능한가?

### 3. 비지원 브라우저 처리
- [ ] Firefox에서 "폴더 연결" 클릭 시 안내 메시지가 뜨는가?
- [ ] Safari에서 "폴더 연결" 클릭 시 안내 메시지가 뜨는가?
- [ ] 안내 메시지에 Chrome/Edge 사용 권장이 포함되어 있는가?

### 4. UI 상태 표시
- [ ] 폴더 연결 완료 시 "연결됨" 상태가 UI에 표시되는가?
- [ ] 연결 해제 기능이 정상 작동하는가?
- [ ] 연결 실패/취소 시 적절한 에러 처리가 되는가?

### 5. 보안 검증
- [ ] HTTPS 환경에서만 작동하는가?
- [ ] 사용자 클릭 없이 자동 호출이 불가능한가?
- [ ] 민감 디렉토리(시스템 폴더 등) 접근이 차단되는가?

### 6. 기존 기능 회귀 검증
- [ ] 폴더 연결 없이도 기존 기능 (프로젝트 등록, 완료 처리)이 정상인가?
- [ ] SAL Grid Viewer가 정상 작동하는가?
- [ ] 콘솔 에러 없음

## 통과 기준

- Chrome/Edge에서 폴더 연결 → 파일 읽기/쓰기 정상 작동
- 핸들 저장/복원으로 재방문 시 폴더 재선택 불필요
- 비지원 브라우저에서 적절한 안내 메시지 표시
- 기존 기능에 영향 없음
