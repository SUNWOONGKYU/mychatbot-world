## 핵심 워크플로우

### 1. 챗봇 생성 워크플로우

```
[사용자 음성 입력]
    → STT 변환 (api/External/tts-proxy.js)
    → AI 분석 (OpenRouter → 업종/타겟/특성 추출)
    → 페르소나 생성 (mcw_personas 테이블)
    → FAQ 자동 생성
    → 챗봇 등록 (mcw_bots 테이블)
    → 배포 (URL + QR 생성)
```

### 2. AI 대화 워크플로우

```
[사용자 메시지 입력]
    → 감성 슬라이더 값 확인 (1~100)
    → 비용 조건 확인 (0원~상한)
    → AI 라우팅 (최적 모델 선택)
    → 페르소나 컨텍스트 로딩
    → KB 검색 (RAG)
    → AI 응답 생성
    → 대화 로그 저장 (mcw_chat_logs)
    → TTS 변환 (선택)
```

### 3. 스킬 실행 워크플로우

```
[스킬 선택]
    → 스킬 메타데이터 로딩
    → 입력 파라미터 수집
    → 스킬 런타임 실행 (프롬프트 기반)
    → 결과 반환
    → 사용량 기록
```

### 4. 수익활동 워크플로우

```
[의뢰 등록]
    → AI 매칭 (챗봇 역량 ↔ 의뢰 요건)
    → 후보 알림
    → 채용 확정
    → 작업 수행
    → 결과 검수
    → 수수료 정산 (20%)
    → 리뷰 등록
```

## 시스템 워크플로우

### 멀티채널 배포

```
[챗봇 생성 완료]
    → URL 배포 (mychatbot.world/bot/{id})
    → QR 코드 생성
    → 임베딩 코드 생성
    → 텔레그램 연동 (선택)
    → Obsidian 연동 (선택)
    → CPC 원격 실행 (선택)
```
