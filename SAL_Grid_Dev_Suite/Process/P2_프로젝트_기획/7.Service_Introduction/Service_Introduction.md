## 핵심 특징

### 1. 5분 생성
음성 인터뷰만으로 챗봇 자동 생성. 코딩 불필요.

### 2. 5단계 라이프사이클
생성 → 학습 → 성장 → 수익활동 → 피상속

### 3. 순수 개인 5대 고객군 특화
일반 개인·소상공인·전문직·프리랜서·정치인 (기업 B2B 제외)

### 4. 멀티 AI 라우팅
감성 슬라이더 + 비용 슬라이더 → 최적 AI 자동 선택

### 5. 스킬 마켓 생태계
누구나 스킬을 만들고 판매 — 챗봇 능력 무한 확장

### 6. 디지털 유산 상속
사전 동의 기반 피상속 — 업계 최초

### 7. 4축 수익모델
구독(Free/Basic/Pro) + 수수료(20%) + API 사용료(마진30%) + 템플릿(30만원/건)

## 6대 챗봇 유형

| # | 유형 | 타겟 고객군 | 핵심 기능 |
|---|------|-----------|---------|
| 1 | 대리형 아바타 (소상공인용) | 카페·식당·학원·병원·미용실·온라인 판매자 | 24시간 고객 문의·예약·상담 자동 대리 |
| 2 | 대리형 아바타 (정치인용) | 국회의원·지자체장·지방의원·출마 예정자 | 유권자 문의·공약 안내·민원 대응 자동화 |
| 3 | 수익형 아바타 | 변호사·세무사·의사·프리랜서·컨설턴트 | 1차 상담 자동화 + 유료 서비스 연결 |
| 4 | 부업형 아바타 | 취미·재능 보유 일반인 | 취미·노하우 공유 + 소액 수익 창출 |
| 5 | 전문 실행형 도우미 | 5대 고객군 전체 | 업무대행·투자자동매매·AI 원격 실행 |
| 6 | 생활 도우미 | AI 소외 일반 국민 | 일상 전반 AI 지원 (무료·저비용 중심) |
