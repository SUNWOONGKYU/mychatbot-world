## 4축 수익 구조

### 1축: 구독 수익

| 플랜 | 월 요금 | 서비스 |
|------|--------|--------|
| Free | 0원 | 챗봇 1개, 기본 AI, 기능 제한 |
| Basic | 19,900원 | 챗봇 3개, 기본 AI, 스킬 마켓 |
| Pro | 49,900원 | 챗봇 10개, 프리미엄 AI, 스킬 마켓, 수익활동, 멀티채널 |

### 2축: 수수료 수익 (20% 통일)

| 항목 | 수수료율 |
|------|---------|
| 스킬 마켓 거래 | 20% |
| 수익활동 중개 | 20% |

### 3축: API 사용료 (종량제 + 마진 30%)

| 항목 | 과금 방식 | 마진율 |
|------|---------|--------|
| AI API 호출 | 종량제 (챗봇 대화·생성·분석) | 30% |

### 4축: 템플릿 판매

| 항목 | 금액 |
|------|------|
| 직종별 맞춤 템플릿 | 30만원/건 |

## 수익 흐름도

```
사용자 → [무료 체험] → [유료 구독] ──────→ 1축 구독 수익
                            ↓
                    [스킬 장착/구매] ────→ 2축 수수료 (20%)
                    [수익활동 참여] ────→ 2축 수수료 (20%)
                    [AI API 사용] ──────→ 3축 API 사용료 (마진 30%)
                    [템플릿 구매] ──────→ 4축 30만원/건
```
