## 1. 사업 개요

### 1.1 서비스명
**My Chatbot World** (마이챗봇월드)

### 1.2 도메인
- 메인: `mychatbot.world`
- 대안: `mychatbot.kr`, `mychatbot.io`

### 1.3 핵심 컨셉
> "당신의 AI 챗봇이 이 세상에 태어나서
> 당신에 대해 공부하고
> 당신과 대화하며 성장합니다
>
> 그래서 세상 곳곳에 나가
> 당신을 대신해서 24시간 일합니다"

### 1.4 타겟 고객
1. **정치인** (국회의원, 지방의원, 당대표)
2. **크리에이터** (유튜버, 인플루언서)
3. **기업가** (CEO, 창업자, 프리랜서)
4. **전문가** (강사, 컨설턴트, 변호사)
5. **자영업** (식당, 카페, 쇼핑몰)

### 1.5 서비스 정의
개인과 소상공인을 위한 **AI 챗봇 생성 및 배포 플랫폼**

- 음성 3분으로 챗봇 생성
- 분야별 전문 템플릿 제공
- 100+ 무료 스킬 마켓플레이스
- 멀티채널 배포 (명함, 홈페이지, SNS, QR 코드)

## 2. 시장 분석

### 2.1 글로벌 시장

#### 시장 규모
- **2023년**: $5.1B
- **2028년 예상**: $15.5B
- **연평균 성장률 (CAGR)**: 25%

#### 주요 경쟁사

**Tier 1: 엔터프라이즈급**
| 서비스 | 가격 | 특징 |
|--------|------|------|
| Intercom | $74~$395/월 | 기업 고객지원 |
| Drift | $2,500/월 | B2B 세일즈 |
| LivePerson | 맞춤 견적 | 대기업용 |

**Tier 2: 중소기업급**
| 서비스 | 가격 | 특징 |
|--------|------|------|
| ManyChat | $15~$145/월 | 소셜미디어 마케팅 |
| Chatfuel | $15~$300/월 | 페이스북 메신저 |
| Landbot | $40~$400/월 | 시각적 빌더 |
| Tidio | $29~$749/월 | 이커머스 |

**Tier 3: 스타트업/개인**
| 서비스 | 가격 | 특징 |
|--------|------|------|
| Botpress | 무료~유료 | 오픈소스 |
| Chatbase | $19~$99/월 | 간단한 빌더 |

#### 시장 특징
- **80% 이상**이 기업용 (B2B)
- **개인/소상공인 시장** 미개척
- 복잡한 설정이 진입장벽

### 2.2 한국 시장

#### 현황
- **기업용 챗봇**: 카카오, 네이버 독점
- **개인용 챗봇**: 거의 전무
- 대부분 **대행사 방식** (비용 수백만원~수천만원)

#### 기회
- ✅ **블루오션** - 개인 맞춤 챗봇 서비스 없음
- ✅ **한국어 특화** 필요
- ✅ **모바일 최적화** 필요
- ✅ **카카오톡 연동** 수요 높음

### 2.3 경쟁 분석

#### 기존 서비스의 문제점
1. **복잡함**: 폼 입력 30개, 설정 시간 30분~1시간
2. **비쌈**: 월 $15~500 (2만원~70만원)
3. **범용적**: 특정 분야 특화 없음
4. **확장 어려움**: 고정 기능, 커스터마이징 제한

#### 우리의 기회
- **극단적 단순화**: 음성 3분
- **저렴한 가격**: 5만원/월
- **분야 특화**: 템플릿 제공
- **확장 가능**: 스킬 마켓

## 3. 핵심 차별점

### 3.1 차별점 ①: 음성 3분으로 탄생 🎤

#### 기존 방식
```
1. 회원가입
2. 챗봇 이름 입력
3. 역할 선택
4. 인사말 작성
5. FAQ 10개 입력
6. 말투 설정
7. 카테고리 설정
8. 디자인 설정
9. 테스트
10. 배포
━━━━━━━━━━━━━━━━━━━━━
총 소요시간: 30분~1시간
```

#### 우리 방식
```
1. 회원가입
2. 🔴 마이크 버튼 클릭
3. 3분 동안 자유롭게 말하기
   "안녕하세요, 저는 홍길동이고요
    서울 강남구 국회의원입니다.
    교육과 복지 정책을 주로 다루고..."
4. AI가 자동으로 챗봇 생성
   ✅ 이름 추출
   ✅ 역할 파악
   ✅ 전문분야 인식
   ✅ 말투 분석
   ✅ 인사말 자동 생성
   ✅ 기본 FAQ 자동 생성
5. 완성!
━━━━━━━━━━━━━━━━━━━━━
총 소요시간: 3~5분
```

#### 기술 구현
```
음성 입력 (사용자 3분)
    ↓
OpenAI Whisper API
    → 음성을 텍스트로 변환
    ↓
GPT-4o API
    → 정보 추출 및 분석
    {
      "name": "홍길동",
      "role": "국회의원",
      "expertise": ["교육", "복지"],
      "tone": "정중하고 친근함",
      "greeting": "안녕하세요! 저는...",
      "faqs": [...]
    }
    ↓
자동으로 챗봇 생성
```

#### 차별화 포인트
- ✅ **세계 최초** 음성 기반 챗봇 생성
- ✅ **진입장벽 제거** (타이핑 제로)
- ✅ **바이럴 가능성** ("말만 하면 된다고?")
