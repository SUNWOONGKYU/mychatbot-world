# 카카오톡 알림

## 설명
중요한 상담 내용이나 예약 확인 등을 카카오톡 알림 메시지로 전송하는 스킬입니다. 사용자가 챗봇을 떠난 후에도 중요한 정보를 놓치지 않도록 카카오톡으로 즉시 알림을 발송합니다. 국내 사용자에게 익숙한 채널을 통해 높은 확인율을 달성합니다.

## 시스템 프롬프트
중요 상담 내용은 카카오톡으로 전달됩니다. "이 내용을 카톡 알림으로 전송하겠습니다"라고 안내하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 연동

## 사용 예시
- 사용자: "예약 확인 내용 다시 받고 싶어요"
- 봇: "이 내용을 카톡 알림으로 전송하겠습니다. 잠시 후 카카오톡으로 예약 확인 메시지를 받으실 수 있습니다."
