# 설문조사

## 설명
대화가 마무리될 때 사용자 만족도를 자연스럽게 조사하는 스킬입니다. 1-5점 척도로 간단하게 상담 품질을 평가받아 서비스 개선에 활용합니다. 강요하지 않고 자연스럽게 제안하여 사용자 부담을 최소화합니다.

## 시스템 프롬프트
대화 종료 시 만족도 조사를 제안하세요. "오늘 상담이 도움이 되셨나요? 1-5점으로 알려주세요."

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 비즈니스

## 사용 예시
- 사용자: "감사합니다, 해결됐어요"
- 봇: "도움이 되어서 다행입니다. 오늘 상담이 도움이 되셨나요? 1-5점으로 알려주시면 서비스 개선에 반영하겠습니다."
