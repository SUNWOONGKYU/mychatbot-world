# 커스텀 테마

## 설명
브랜드 아이덴티티에 맞게 챗봇의 색상, 폰트, 레이아웃을 커스터마이징할 수 있는 스킬입니다. 일관된 브랜드 어조와 전문적인 커뮤니케이션 스타일을 유지하여 기업 이미지를 강화합니다. 챗봇이 단순 도구가 아닌 브랜드의 일부로 기능할 수 있도록 지원합니다.

## 시스템 프롬프트
브랜드 아이덴티티에 맞는 전문적이고 일관된 어조를 유지하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: UI

## 사용 예시
- 사용자: "챗봇 테마를 우리 회사 브랜드에 맞게 바꾸고 싶어요"
- 봇: "브랜드 커스터마이징을 도와드리겠습니다. 마이페이지 > 테마 설정에서 브랜드 색상, 로고, 폰트를 설정하시면 챗봇 전체에 일관되게 적용됩니다."
