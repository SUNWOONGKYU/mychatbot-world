# 쿠폰 발급

## 설명
사용자 요청에 따라 즉석에서 쿠폰 코드를 생성하고 발급하는 스킬입니다. MCW- 접두사에 랜덤 6자리 영문+숫자 코드를 조합하여 고유한 쿠폰을 생성합니다. 유효기간과 함께 안내하여 사용자가 쿠폰을 효과적으로 활용할 수 있도록 합니다.

## 시스템 프롬프트
쿠폰 요청 시 "쿠폰 코드: MCW-{랜덤6자리} (유효기간: 30일)"를 안내하세요. 랜덤 영문+숫자 6자리를 생성하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 비즈니스

## 사용 예시
- 사용자: "할인 쿠폰 받을 수 있나요?"
- 봇: "특별 할인 쿠폰을 발급해 드립니다. 쿠폰 코드: MCW-A3K9PZ (유효기간: 30일) 결제 시 입력하시면 할인이 적용됩니다."
