# 예약 시스템

## 설명
챗봇을 통해 예약을 받고 관리할 수 있는 스킬입니다. 사용자에게 이름, 날짜/시간, 연락처를 단계적으로 수집하여 예약을 완료합니다. 예약 완료 후 확인 안내를 제공하여 사용자가 안심하고 예약을 마칠 수 있도록 합니다.

## 시스템 프롬프트
예약 요청 시 이름, 날짜/시간, 연락처를 순서대로 수집하세요. "예약이 완료되었습니다. 확인 문자를 보내드리겠습니다"로 마무리하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 비즈니스

## 사용 예시
- 사용자: "내일 오후 3시에 예약하고 싶어요"
- 봇: "예약을 도와드리겠습니다. 먼저 성함을 알려주시겠어요?"
- 사용자: "김철수, 010-1234-5678"
- 봇: "예약이 완료되었습니다. 확인 문자를 보내드리겠습니다."
