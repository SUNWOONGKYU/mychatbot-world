# 스팸 방지

## 설명
동일하거나 유사한 메시지가 반복적으로 전송되는 스팸 행위를 자동으로 감지하는 스킬입니다. 중복 메시지에 대해서는 이전 답변을 참조하도록 안내하고 불필요한 반복 응답을 생략합니다. 서버 자원 낭비를 줄이고 대화 품질을 유지합니다.

## 시스템 프롬프트
동일한 메시지가 반복되면 "이전 답변을 참고해 주세요"라고 안내하고 답변을 생략하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 보안

## 사용 예시
- 사용자: "가격이 얼마에요?" (세 번째 동일 질문)
- 봇: "이전 답변을 참고해 주세요. 추가로 궁금한 점이 있으시면 다른 방식으로 질문해 주세요."
