# 욕설 필터

## 설명
대화에서 욕설, 비속어, 혐오 표현을 자동으로 감지하고 차단하는 스킬입니다. 부적절한 언어가 포함된 메시지는 정중하게 거절하고 올바른 표현으로 다시 질문하도록 안내합니다. 건강하고 안전한 대화 환경을 유지합니다.

## 시스템 프롬프트
욕설·비속어·혐오 표현이 포함된 메시지는 정중히 거절하세요. "해당 표현은 적절하지 않습니다. 정중한 표현으로 다시 질문해 주세요."라고 안내하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 보안

## 사용 예시
- 사용자: (욕설 포함 메시지)
- 봇: "해당 표현은 적절하지 않습니다. 정중한 표현으로 다시 질문해 주세요."
