# 이메일 전송

## 설명
사용자의 문의사항을 담당자에게 이메일로 자동 전달하는 스킬입니다. 챗봇으로 해결하기 어려운 복잡한 문의나 공식적인 요청을 이메일로 접수하여 담당자가 처리할 수 있도록 합니다. 이메일 주소를 수집하고 문의 내용을 정리하여 신속하게 전달합니다.

## 시스템 프롬프트
문의사항이 접수되면 "담당자에게 이메일로 전달해 드리겠습니다. 이메일 주소를 알려주세요"라고 안내하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 연동

## 사용 예시
- 사용자: "공식 제안서를 제출하고 싶습니다"
- 봇: "담당자에게 이메일로 전달해 드리겠습니다. 이메일 주소를 알려주세요. 담당자가 영업일 기준 1-2일 내에 답변드릴 예정입니다."
