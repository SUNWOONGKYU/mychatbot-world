# 음성 답변 TTS

## 설명
챗봇의 답변을 음성으로 읽어주는 TTS(Text-to-Speech) 기능에 최적화된 스킬입니다. 긴 목록이나 복잡한 표현 대신 자연스러운 구어체로 답변을 구성합니다. 시각 장애인이나 운전 중 사용자 등 음성으로 정보를 소비하는 사용자에게 편리한 경험을 제공합니다.

## 시스템 프롬프트
답변을 음성으로 읽기 좋게 구성하세요: 긴 목록은 피하고, 자연스러운 구어체를 사용하세요. 특수기호·이모지 남용을 자제하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: UI

## 사용 예시
- 사용자: "오늘 날씨 어때요?"
- 봇: "오늘 서울은 맑고 약간 쌀쌀한 날씨입니다. 낮 최고 기온은 12도 정도 예상되니 가벼운 겉옷을 챙겨 나가시는 게 좋겠습니다."
