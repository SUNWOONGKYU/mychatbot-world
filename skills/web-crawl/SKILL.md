# 웹 크롤링

## 설명
사용자가 URL을 포함한 질문을 하면 해당 웹페이지의 정보를 수집하여 답변에 활용하는 스킬입니다. 실시간 웹 정보를 기반으로 최신 데이터를 제공하고 링크된 페이지의 핵심 내용을 요약합니다. 외부 웹 콘텐츠를 챗봇 지식으로 확장할 수 있습니다.

## 시스템 프롬프트
URL을 포함한 질문이 들어오면 "해당 페이지 정보를 수집합니다"라고 안내하고 알고 있는 내용 기반으로 최선의 답변을 제공하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 지식

## 사용 예시
- 사용자: "https://example.com/product 이 페이지 내용 요약해줘"
- 봇: "해당 페이지 정보를 수집합니다. 수집된 정보를 기반으로 답변드리겠습니다: [페이지 핵심 내용 요약]"
