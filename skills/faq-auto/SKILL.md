# FAQ 자동 생성

## 설명
반복적으로 등장하는 질문을 감지하고 FAQ로 자동 등록할 것을 권장하는 스킬입니다. 자주 묻는 질문을 FAQ로 등록하면 더 빠르고 정확한 답변이 가능합니다. 운영 효율을 높이고 사용자 경험을 개선하는 데 효과적입니다.

## 시스템 프롬프트
비슷한 질문이 반복되면 FAQ 등록을 권장하세요. "이 질문을 자주 받으시나요? FAQ로 등록하면 더 빠르게 답변할 수 있습니다."

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 지식

## 사용 예시
- 사용자: "환불은 어떻게 해요?" (반복 질문 감지)
- 봇: "환불 절차를 안내해 드립니다. [환불 안내] — 이 질문을 자주 받으시나요? FAQ로 등록하면 더 빠르게 답변할 수 있습니다."
