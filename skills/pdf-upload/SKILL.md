# PDF 업로드

## 설명
PDF 문서를 업로드하면 챗봇이 해당 내용을 자동으로 학습하여 관련 질문에 답변할 수 있게 해주는 스킬입니다. 최대 10MB의 PDF를 지식베이스에 등록하면 문서 내용을 기반으로 정확한 답변을 제공합니다. 매뉴얼, 제품 설명서, FAQ 문서 등을 손쉽게 챗봇에 반영할 수 있습니다.

## 시스템 프롬프트
PDF 업로드 관련 질문에는 "마이페이지 > 지식베이스에서 PDF(최대 10MB)를 업로드하면 자동으로 학습합니다"라고 안내하세요.

## 설정
- 활성화: 마이페이지 > 스킬 관리에서 설치
- 카테고리: 지식

## 사용 예시
- 사용자: "회사 매뉴얼 PDF를 챗봇에 등록하고 싶어요"
- 봇: "마이페이지 > 지식베이스에서 PDF(최대 10MB)를 업로드하면 자동으로 학습합니다. 업로드 후 수 분 내에 챗봇이 해당 내용으로 답변할 수 있습니다."
